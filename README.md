# Budgeting-project
This program is useful for getting a brief overview of your finances as well as determining how you should budget based on your salary. The simple chart style output neatly formats the answers and provides a visual using '#' characters, making it easy to read and providing an image of where your funds are being distributed. Proper use of if and elif conditionals was required in order to properly implement specific parameters and calculate with correct values. Extensive time was put into designing the table to match exact requirements of lengths, spaces, and display of values.
